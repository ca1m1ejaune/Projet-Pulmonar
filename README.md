## 📖 Présentation

**Projet Pulmonar** est un projet de vision par ordinateur visant à analyser et interpréter des images médicales pulmonaires.  
Ce dépôt contient l'ensemble du code source permettant de charger, prétraiter et analyser ces données dans un cadre de recherche ou d'application médicale.

---

## 🧑‍💻 Technologies utilisées

- **Python** (65.6%)  
  Scripts pour la manipulation de données, l'entraînement de modèles et l'évaluation.
- **JavaScript** (26%)  
  Composants interactifs pour l'interface web et la visualisation des résultats.
- **CSS** & **HTML** (8.4%)  
  Mise en forme et structure de l'interface utilisateur.

---

## 🚀 Fonctionnalités principales

- Chargement d'images médicales (ex : radiographies pulmonaires)
- Prétraitement automatique (redimensionnement, normalisation, etc.)
- Entraînement de réseaux de neurones (CNN, etc.) pour la détection de pathologies pulmonaires
- Interface web pour la visualisation des résultats, la prédiction en ligne et le partage des analyses

## ⚡ Comment démarrer

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/ca1m1ejaune/Projet-Pulmonar.git
   ```
2. **Configurer l'environnement Python**
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
3. **Lancer l’application**
   - Backend IA :
     ```bash
     python main.py
     ```
   - Frontend :
     ```bash
     cd ../frontend
     # Instructions selon le framework JS utilisé (ex: npm start)
     ```
